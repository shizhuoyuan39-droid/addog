1. 프로젝트 개요 / 项目概述
한국어
본 모듈은 Flutter 여행 앱 내 웨이하이시(威海市) 관광 정보 데이터를 정의한 소스 파일입니다. 도시 기본 정보, 관광지, 특산 음식, 3일 여행 코스, 바로가기 메뉴, AI 응답 규칙 등 모든 콘텐츠를 통합 관리하며, 앱 내 지역별 여행 페이지에 연동하여 사용합니다.
中文
本模块是 Flutter 旅行应用中威海市的旅游信息数据文件。整合了城市基础资料、景点、特色美食、三日游玩路线、快捷入口、智能问答规则等全部内容，对接应用内各地旅行专题页面使用。
2. 파일 정보 / 文件说明
한국어
파일 경로: lib/models/city_data.dart
주요 종속 파일: travel_models.dart (데이터 모델 정의)
데이터 타입: CityTravelData 상수 객체 (weihaiCity)
中文
文件路径：lib/models/city_data.dart
依赖文件：travel_models.dart（基础数据模型定义）
数据类型：CityTravelData 常量对象（变量名：weihaiCity）
3. 사용 방법 / 使用方式
한국어
제공된 코드를 lib/models/city_data.dart 파일에 전체 복사하여 붙여넣습니다.
앱 내 cities 리스트(도시 목록)에 weihaiCity 변수를 등록합니다.
앱 재실행 후 웨이하이시 관광 페이지가 정상 로드되는지 확인합니다.
中文
将完整代码复制粘贴至 lib/models/city_data.dart 文件中。
在项目的 cities 城市列表里注册 weihaiCity 变量。
重启应用，验证威海专题页面可正常展示所有内容。
4. 데이터 구성 항목 / 数据结构说明
한국어
weihaiCity 객체는 다음 카테고리로 나뉘어 데이터를 구성합니다.
기본 도시 정보
ID, 성/시 명칭, 약칭, 모듈 제목, 메인 타이틀, 배경 이미지, 도시 소개(위치/특징/기후 등)
주요 관광지(attractions)
류궁다오, 웨이하이 국제해수욕장, 화쥐바제 정보 (이미지, 설명, 주요 볼거리, 주소, 운영시간 포함)
특색 음식(foods)
해산물 찜, 고등어 물만두, 해초 냉채 (맛과 설명 포함)
3일 여행 일정(itinerary)
1~3일차 상세 동선, 시간, 소요시간, 유의사항으로 구성된 완전 여행 코스
퀵 메뉴(quickEntries)
도시소개, 관광지, 미식, AI 문답 4가지 바로가기 아이콘 및 스타일
AI 여행 문답(AI 관련 필드)
인사말, 키워드별 자동 응답, 기본 답변 규칙 설정
中文
weihaiCity 对象分为以下板块存储数据：
城市基础信息
城市 ID、省市名称、简称、模块标题、首页标题、轮播图、城市综合介绍（地理位置 / 城市简介 / 气候 / 特色）
热门景点 (attractions)
刘公岛、威海国际海水浴场、火炬八街（含配图、简介、游玩亮点、地址、开放说明）
特色美食 (foods)
海鲜蒸锅、鲅鱼水饺、凉拌海菜（口味与菜品介绍）
三日行程 (itinerary)
完整 3 天旅行路线，包含时间、地点、时长、游玩小贴士
快捷入口 (quickEntries)
城市介绍、景点、美食、智能问答四大功能入口，配置图标与主题色
智能问答 (AI 相关字段)
欢迎语、关键词自动回复、默认回复规则

한국어
